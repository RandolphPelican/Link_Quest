# ⚔️ Link Quest

A browser-based top-down multiplayer RPG built by a dad and his kids —
to prove that AI-assisted coding isn't a cop-out. It's a superpower.

**Play now:** https://randolphpelican.github.io/Link_Quest/

## 🎮 Controls

- **WASD / Arrow Keys** — Move
- **SPACE** — Attack
- **E** — Cast spell
- **F** — Open chests / Read signs
- **P** — Pause

## 👾 Enemy Types

**Level 1:** Goblin, Goblin Chief, AI Bug, Chatbot Clone
**Level 2:** Glitch Sprite, Memory Leak, Phantom Var, Null Pointer
**Level 3:** Stack Overflow, Syntax Error, Dark Compiler, Trojan Horse

Find them all.

## 🚀 Run Locally

```bash
git clone https://github.com/RandolphPelican/Link_Quest.git
cd Link_Quest
python3 -m http.server 8080
```

Open http://localhost:8080 in your browser.

## 🛠️ Stack

- Pure Canvas2D — zero dependencies, runs everywhere
- Custom game engine (engine.js)
- Vanilla JS — zero build step, zero npm
