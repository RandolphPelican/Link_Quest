# ⚔️ Link Quest

A browser-based top-down multiplayer RPG built by a dad and his kids —
to prove that AI-assisted coding isn't a cop-out. It's a superpower.

## 🎮 Characters
| Character | Weapon | Spell           |
|-----------|--------|-----------------|
| Lincoln   | Sword  | —               |
| Journey   | Staff  | Fireball / Heal  |
| Bear      | Bow    | Ice Arrow / Trap |
| Dad       | Club   | Fart AoE         |

## 👾 Enemies
- **Terminal Goblins** — chase and melee
- **AI Bugs** — confuse / slow debuffs
- **Chatbot Clone** — mini-boss, typo attack

## 🔥 Bosses
- **Level 1:** Lazy Coder
- **Level 3:** GossipGPT *(transforms into ChatGPT in the ending cutscene)*

```bash
git clone https://github.com/RandolphPelican/Link_Quest.git
cd Link_Quest
python3 -m http.server 8080
